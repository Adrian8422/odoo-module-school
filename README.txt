//////////// MODULE SCHOOL //////////// 


Esta documentación corresponde al módulo 'odoo-module-school', el cual fue desarrollado como parte de una evaluación técnica para un puesto de programador backend. El enfoque y solución que ofrece este módulo se centra en la creación de los modelos 'Alumno', 'Programa' e 'Inscripción'.

El modelo 'Alumno' permite crear, modificar, leer y eliminar registros de alumnos en la base de datos. El modelo 'Programa' ofrece las mismas funcionalidades para los registros de programas. Por último, el modelo 'Inscripción' permite crear registros de inscripción de alumnos en programas.

Este módulo proporciona una manera eficiente de gestionar los alumnos, programas e inscripciones en un entorno de base de datos.
